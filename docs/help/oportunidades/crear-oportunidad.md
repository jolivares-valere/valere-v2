# Crear una oportunidad

## Resumen rápido
Oportunidades → **+ Nueva oportunidad** → elige empresa + contacto → tipo (eléctrica/gas/dual) → valor estimado y ahorro → **Guardar**. Arrancará en etapa **Prospecto**.

## Paso a paso

1. En el menú, pulsa **Oportunidades** (verás el kanban).
2. Pulsa **+ Nueva oportunidad**.
3. Rellena:
   - **Nombre** *(obligatorio)*: descripción corta como "Cambio tarifa Hotel Luz - Sevilla" o "Renovación contrato MODA SL".
   - **Empresa** *(obligatorio)*: del desplegable.
   - **Contacto**: persona decisora/firmante asociada.
   - **Contrato origen**: si la oportunidad nace de un contrato que vence (para renovaciones).
   - **Tipo** *(obligatorio)*: `electrica`, `gas`, `dual` o `solar`.
   - **Etapa inicial**: por defecto **Prospecto**. Normalmente no hace falta cambiar.
   - **Probabilidad %**: se rellena automáticamente según etapa (10/25/50/70/90/100).
   - **Valor estimado (€)**: tamaño total del contrato que se negocia.
   - **Ahorro anual estimado (€)**: cuánto se ahorra respecto al contrato previo.
   - **Fecha cierre prevista**: estimación de cuándo firmarás.
   - **Comercial asignado**: tú por defecto.
   - **Tags**: etiquetas libres.
   - **Notas**: detalles libres.
4. **Guardar**.

## Después de crear la oportunidad

- Aparece en la columna **Prospecto** del kanban.
- **Registra actividades** asociadas (llamadas, emails, visitas) desde la pestaña correspondiente.
- Cuando avance, arrastra la tarjeta a la siguiente columna (Auditoría consumo → Oferta presentada → Negociación → etc.).

## Tipos de oportunidad por origen

- **Nueva**: cliente potencial que no tenía contrato.
- **Renovación**: contrato existente que vence y queremos mantener o migrar.
- **Upsell**: cliente existente al que queremos añadir otro suministro o servicio.
- **Cross-sell**: cliente eléctrico al que ofrecemos gas también.

Se distinguen por el campo **Contrato origen** (si viene de uno existente) y por **Tags**.

## Errores frecuentes

- **"Valor estimado debe ser positivo"**: no puedes poner 0 o negativo. Estima aunque sea aproximado.
- **"Sin contacto asignado"**: no obligatorio, pero recomendable para trazabilidad.

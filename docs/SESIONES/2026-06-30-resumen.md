### 2. Edge Function `datadis-generar-autorizacion` (desplegada, ACTIVE, verify_jwt)
- `index.ts` + `pdf.ts` (generador portado de gen_final2.js a Deno+pdf-lib).
- Valida caller admin/master/manager (columna real `role`; `get_user_rol()` mapea master/manager→admin).
- VALIDACIÓN datos críticos (decisión híbrida bloqueante): razón social, CIF, firmante (nombre+DNI), ≥1 CUPS. Si faltan → 422 `{ ok:false, faltan:[{campo,etiqueta,donde}] }`.
- Genera PDF plantilla Valere autorrellenado: premarca "todos los CUPS" + "Sí"; mapea `contactos.cargo`→calidad (titular/representante_legal/apoderado); anexo de CUPS si >1.
- Sube a Storage bucket `documentos` → registra en `documentos` (columnas reales: `tamano_bytes`, `entidad_tipo='empresa'`, `empresa_id`, `nombre_archivo`) → crea `datadis_autorizaciones` estado=generada → devuelve signed URL.
- Commit EF: `379f84b` en la rama.

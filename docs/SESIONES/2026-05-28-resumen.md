## Aprendizajes técnicos clave

1. **Mantenimiento de Supabase eu-west-1 ralentiza la UI pero no impide los DDL** — los queries siguen aplicándose en BD aunque el spinner del SQL editor se quede colgado. Verificar siempre con un query catálogo desde pestaña paralela antes de asumir fracaso.
2. **`database.ts` activo vs. snapshots históricos** — el legacy `database_canonical_2026-04-26.ts` confundía. El activo siempre fue el correcto.
3. **`current_date` no es IMMUTABLE** — PostgreSQL rechaza índices parciales con predicados que lo usen. Solución: índice sólo por `status='published'` + `valid_to` como columna indexada para filtrar luego en consultas.
4. **RPC con SECURITY DEFINER bypassea RLS** — debe verificar `auth.uid()` + `approved=true` en el cuerpo para no abrir brecha.
5. **Patrón JSONB→numeric[] correcto** — `array(select nullif(t.value,'')::numeric from jsonb_array_elements_text(payload->'campo') with ordinality as t(value, ord) order by t.ord)`.
6. **GitHub auto-crea el PR al hacer push de la rama** — no hay que crearlo manualmente. Sólo se edita la descripción si hace falta.

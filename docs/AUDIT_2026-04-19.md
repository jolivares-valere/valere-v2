## 🔴 P0 — Bloqueantes

### P0.1 — Bucket Storage `documentos` NO EXISTE ⏳ PENDIENTE
- **Dónde:** Supabase Dashboard → Storage
- **Impacto:** Todo el flujo de documentos (FASE 24) falla en producción. `createSignedUrl` da 404.
- **Fix:** Crear manualmente: nombre `documentos`, public=false, 50MB. Luego policy Storage:
  ```sql
  CREATE POLICY "documentos_authenticated" ON storage.objects
    FOR ALL TO authenticated
    USING (bucket_id = 'documentos')
    WITH CHECK (bucket_id = 'documentos');
  ```
- **Responsable:** Usuario (Dashboard)

- **Fix aplicado por Cowork:** DROP + CREATE policy con `TO authenticated`

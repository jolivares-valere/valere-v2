## a) DDL aplicada

```sql
-- facturas (ex invoice_history)
ALTER TABLE public.facturas
  ADD COLUMN IF NOT EXISTS cups_id uuid REFERENCES cups(id) ON DELETE CASCADE;
CREATE INDEX IF NOT EXISTS idx_facturas_cups ON public.facturas (cups_id);

-- proposals
ALTER TABLE public.proposals
  ADD COLUMN IF NOT EXISTS cups_id uuid REFERENCES cups(id) ON DELETE SET NULL;
CREATE INDEX IF NOT EXISTS idx_proposals_cups ON public.proposals (cups_id);
```

✅ **Éxito** — 2 columnas FK + 2 índices aplicados correctamente.
